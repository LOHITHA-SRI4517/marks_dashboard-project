# 📊 Student Performance Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard to analyze student performance across subjects like Mathematics, Physics, and Chemistry. It provides meaningful insights into academic performance using data visualization techniques.

---

## 🚀 Key Features
- 📊 Subject-wise marks analysis
- 📈 Percentage distribution insights
- 👩‍🎓 Student-wise performance comparison
- 📉 Identification of top and low performers
- 📅 Trend analysis across subjects
- 🎯 Interactive filters for dynamic exploration

---

## 🛠️ Tools & Technologies
- Power BI  
- MS Excel
